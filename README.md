## Descripción

Programa desarrollado en C# que permite registrar una cantidad de estudiantes, ingresar cuatro calificaciones por cada uno, calcular el promedio y determinar si el estudiante aprobó o reprobó. Los resultados se muestran organizados en formato de tabla.

## Entrada

- Cantidad de estudiantes.
- Nombre del estudiante.
- Cuatro calificaciones.

## Proceso

Se utiliza un ciclo `for` para registrar los datos, calcular el promedio y determinar el estatus de cada estudiante.

## Salida

- Nombre.
- Cuatro calificaciones.
- Promedio.
- Estatus (Aprobado o Reprobado).
